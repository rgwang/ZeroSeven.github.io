高级语言：

  • 按执行方式：
    • 编译型：整个代码一次性进行翻译，执行时是一个整体在执行
    • 解释型：执行时是一句一句在执行
  • 按类型声明：
    • 动态语言：变量可以随时定义随时用，不需要事先声明其是什么类型
    • 静态语言：需要事先声明该变量是什么类型

python属于解释型、动态语言，所以相比执行起来比较慢

第二部分 变量

1.变量的概念

  • “量”：实实在在的对象
  • “变”：可变性，增删查改等
  • 变量定义而要素：变量名、赋值

2. 变量的命名

2.1哪些可以用来做变量名

  • 大写字母、小写字母、数字、下划线、汉字及其组合
  • 严格区分大小写
  • 首字符不能是数字
  File "<ipython-input-11-e3b1d93d01a0>", line 1
    1_fruit = "apple"
     ^
SyntaxError: invalid token
  • 变量名中间不能有空格
  • 不能与33个Python保留字相同

2.3 变量名定义技巧

  • 变量名尽可能有实际意义，表征数据的某种特性

  • 用下划线(多用于变量和函数名）。变量名由多个单词组成，用_连接多个单词

  • 驼峰体（多用于类名）。变量名由多个单词组成，单词首字母大写

  • 尽量避免用中文和拼音做变量名

  • 特殊的变量：常量（如Π，e），变量名所有字母均大写

  1. 使用空格
    • 二元运算符两边加一个空格
  • 使用不同优先级的运算符，考虑在最低优先级的运算符周围添加空格

  1. 以上属于PEP8格式指南的部分内容，养成良好编码规范

    • 使代码风格一致
    • 提升代码可读性
  3. 不要死板执行格式规范
    • 项目规范优先
