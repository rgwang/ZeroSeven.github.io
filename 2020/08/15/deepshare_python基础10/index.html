10.1 time库

Python处理时间的标准库

1、获取现在时间

（1）time.localtime() 本地时间

（2）time.gmtime() UTC世界统一时间

北京时间比时间统一时间UTC早8个小时

t_local time.struct_time(tm_year=2020, tm_mon=8, tm_mday=15, tm_hour=0, tm_min=0, tm_sec=29, tm_wday=5, tm_yday=228, tm_isdst=0)
t_UTC time.struct_time(tm_year=2020, tm_mon=8, tm_mday=14, tm_hour=16, tm_min=0, tm_sec=29, tm_wday=4, tm_yday=227, tm_isdst=0)

'Sat Aug 15 00:00:29 2020'

2、时间戳与计时器

（1）time.time()   返回自纪元以来的秒数，记录sleep

（2）time.perf_counter()   随意选取一个时间点，记录现在时间到该时间点的间隔秒数，记录sleep

（3）time.process_time()   随意选取一个时间点，记录现在时间到该时间点的间隔秒数，不记录sleep

perf_counter()精度较time()更高一些

3、格式化

（1）time.strftime 自定义格式化输出

'2020-08-15 Saturday 00:00:34'

4、睡觉觉

（1）time.sleep()

10.2 random库

随机数在计算机应用中十分常见

Python通过random库提供各种伪随机数

基本可以用于除加密解密算法外的大多数工程应用

1、随机种子——seed(a=None)

（1）相同种子会产生相同的随机数

（2）如果不设置随机种子，以系统当前时间为默认值

0.5714025946899135
0.5714025946899135

0.4288890546751146

2、产生随机整数

（1）randint(a, b)——产生[a, b]之间的随机整数

[10, 1, 4, 8, 8, 5, 3, 1, 9, 8]

（2）randrange(a)——产生[0, a)之间的随机整数

[5, 1, 3, 5, 0, 6, 2, 9, 5, 6]

（3）randrange(a, b, step)——产生[a, b)之间以setp为步长的随机整数

[6, 4, 4, 6, 2, 4, 4, 2, 6, 2]

3、产生随机浮点数

Counter({'blue': 3, 'red': 2, 'green': 1})
  • 是字典的一个子类
True
  • 最常见的统计——most_commom(n)
    提供 n 个频率最高的元素和计数

  • 元素展开——elements()
['牛', '牛', '奶', '奶', '奶', '奶', '奶', '找', '刘', '买']
  • 其他一些加减操作

3、deque——双向队列

列表访问数据非常快速

插入和删除操作非常慢——通过移动元素位置来实现

特别是 insert(0, v) 和 pop(0)，在列表开始进行的插入和删除操作

双向队列可以方便的在队列两边高效、快速的增加和删除元素

deque(['c', 'd', 'e'])

deque(['a', 'b', 'c', 'd', 'e', 'f', 'g'])
